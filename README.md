# 🌍 GeoSafe Logistics

### Predict • Optimize • Deliver

🚀 **GeoSafe Logistics** is an AI-powered supply chain platform that predicts geopolitical & operational risks and dynamically optimizes shipment routes in real-time.

Built for the **DP World Hackathon** by **Team Phoenix**.

---

## 📌 Problem Statement

Global supply chains face frequent disruptions due to:

* Geopolitical conflicts
* Natural disasters
* Port congestion
* Labor strikes

Traditional logistics systems are **static and reactive**, focusing only on cost or distance — leading to:

* 🚫 Delayed shipments
* 💸 Increased operational costs
* 📉 Reduced reliability

---

## 💡 Solution Overview

GeoSafe Logistics provides a **proactive and intelligent system** that:

* Predicts risks using AI
* Optimizes routes dynamically
* Simulates real-world disruptions
* Enhances decision-making with live insights

---

## ✨ Key Features

### 🌐 1. Global Risk Prediction

* Risk scoring for regions based on simulated factors
* Visual heatmap (Low / Medium / High risk)

### 🛣️ 2. Dynamic Route Optimization

* Compare:

  * Shortest route
  * Safest route
  * Eco-friendly route
* Real-time rerouting based on disruptions

### 🚨 3. Alerts System

* Live alerts for:

  * Conflicts
  * Weather issues
  * Port congestion
* Severity levels (Low / Medium / High)

### 🔁 4. Digital Twin Simulation

* Run “what-if” scenarios:

  * Port closures
  * Storms
  * Political instability
* View impact on time, cost, and CO₂

### 📦 5. Shipment Tracking

* Track shipments in real-time
* View risk level, route, and status

### 🌱 6. Sustainability Metrics

* CO₂ emissions tracking
* Fuel consumption insights
* Eco-score for routes
* % reduction in emissions

### 🌍 7. Multi-language Support

* Supports:

  * English
  * Hindi
  * Spanish
  * French

## ⚙️ How It Works

1. User selects a shipment
2. System displays current route
3. Risk engine assigns risk score
4. Optimizer suggests better route
5. Alerts trigger on disruptions
6. Simulation shows future scenarios

---

## 🎯 Demo Flow

1. Open Dashboard
2. Select shipment (Mumbai → Rotterdam)
3. View current (risky) route
4. Click **Optimize Route**
5. See safer + lower CO₂ route
6. Trigger disruption (Simulation)
7. Watch automatic rerouting
